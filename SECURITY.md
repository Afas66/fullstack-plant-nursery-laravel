# Security Policy

## Supported Versions

We are committed to ensuring the security of the Greenthumb Nursery application. Security updates will be applied to the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

The project team and community take all security vulnerabilities seriously. Thank you for your efforts to improve the security of our project. We appreciate your responsible disclosure.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them directly to us via email at **[YOUR_EMAIL_ADDRESS]**.

Please include the following details in your report:
- A clear description of the vulnerability.
- Steps to reproduce the issue.
- The potential impact of the vulnerability.
- Any potential mitigations you have considered.

We will make every effort to respond to your report within 48 hours and will work with you to understand and resolve the issue. We will not disclose your name or the vulnerability publicly until a fix has been released.

Thank you for helping keep Greenthumb Nursery secure.
