# Contributing to Greenthumb Nursery

Thank you for your interest in contributing to Greenthumb Nursery! We welcome all contributions, from bug reports to feature suggestions. To ensure a smooth process, please review the following guidelines.

## How to Contribute

1.  **Fork the Repository:** Start by forking the project to your own GitHub account.
2.  **Create a New Branch:** Create a descriptive branch for your changes.
    ```bash
    git checkout -b feature/add-new-plant-search
    ```
3.  **Make Your Changes:** Write your code and make sure it follows the existing project style.
4.  **Test Your Changes:** Ensure that your changes do not break any existing functionality.
5.  **Commit Your Changes:** Write a clear and concise commit message.
    ```bash
    git commit -m "feat: Add search functionality for plants"
    ```
6.  **Push to Your Fork:**
    ```bash
    git push origin feature/add-new-plant-search
    ```
7.  **Submit a Pull Request:** Open a pull request from your fork to the main Greenthumb Nursery repository. Please provide a detailed description of the changes you have made.

## Reporting Bugs

If you find a bug, please create an issue in the GitHub repository. Include the following information:
-   A clear and descriptive title.
-   Steps to reproduce the bug.
-   The expected behavior and what actually happened.
-   Your operating system, PHP version, and browser.

Thank you for helping make this project better!
